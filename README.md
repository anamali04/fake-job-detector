# 🔍 Fake Job Detector

An AI-powered web application that detects fake job postings using Machine Learning and FastAPI.

## 🚀 Features
- Detects fake job postings with 98% accuracy
- Built with Random Forest ML model
- FastAPI backend with REST API
- Clean and responsive frontend

## 🛠️ Tech Stack
- Python, FastAPI, Scikit-learn
- HTML, CSS, JavaScript
- Random Forest Classifier
- TF-IDF Vectorizer

## 📊 Dataset
- Kaggle: Real or Fake Job Postings Dataset
- 17,000+ job postings

## ⚙️ How to Run
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run backend: `uvicorn main:app --reload`
4. Open `frontend/index.html` in browser
